<h1 align="center">Reproductor Musical en C/C++</h1>
<p>Este repositorio alberga mi código fuente y los archivos relacionados con el proyecto final desarrollado en la clase de Laboratorio de Programación 2. El proyecto consiste en un sistema de reproducción musical que simula un reproductor de música, donde los usuarios pueden seleccionar su género musical preferido y recibir una playlist aleatoria.</p>

<h2 align="center">Descripción del Proyecto</h2>
<p>El principal objetivo de este proyecto fue la aplicación avanzada de conceptos de programación, incluyendo la manipulación de archivos para la gestión de listas de reproducción, el uso de estructuras para organizar datos, y la implementación de validaciones para garantizar una experiencia de usuario consistente y libre de errores.</p>
<hr>
<h2 align="center">Características Principales</h2>
<ul>
  <li><b>Manipulación de Archivos:</b> Se emplea la manipulación de archivos para almacenar y organizar información sobre los álbumes y canciones disponibles. Las listas de reproducción se construyen dinámicamente a partir de estos archivos.</li>
  <li><b>Estructuras para Organización:</b> Se utilizan estructuras para modelar de manera efectiva la información sobre géneros, álbumes y canciones, mejorando la organización y modularidad del código.</li>
  <li><b>Colas para Listas de Reproducción:</b> Se implementa una estructura de cola para gestionar la lista de reproducción, asegurando que las canciones se agreguen en el orden adecuado.</li>
  <li><b>Validaciones Avanzadas:</b> Se incorporan validaciones robustas para evitar repeticiones en las selecciones de género y para garantizar la coherencia de la experiencia del usuario.</li>
</ul>
<hr>
<h2 align="center">Instrucciones de Uso</h2>
<ul>
  <li><b>Compilación:</b> Utiliza un compilador de C para compilar el código fuente. Por ejemplo, puedes usar GCC con el siguiente comando: 'gcc reproductormusical.c -o reproductormusical'.</li>
  <li><b>Ejecución:</b> Ejecuta el programa resultante con './reproductormusical' en la línea de comandos.</li>
  <li><b>Selección de Género:</b> Inicia el programa y sigue las instrucciones en la consola para seleccionar tu género musical preferido.</li>
  <li><b>Reproducción de Playlist:</b> Una vez seleccionado el género, se generará automáticamente una playlist aleatoria para tu disfrute.</li>
</ul>
